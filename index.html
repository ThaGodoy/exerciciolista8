1) Para reforçar os conceitos apresentados em aula, faça mapeamento de QTLs para o conjunto de dados MAIZE DATA usando análise de cada marcador (single marker analysis). Use apenas funções do R base, ou seja, não carregue nenhum pacote. Apresente os resultados (incluindo os códigos) considerando que o mapa genético já foi estimado.

Os códigos utilizados no RStudio foram os seguintes:

  • Abrir dados maize 
maize<-read.csv("http://dl.dropbox.com/u/1968009/maize.csv")
  • regressão linear para calcular o efeito aditivo, calculando a diferença entre os homozigotos e o heterozigoto é a média dos homozigostos
testeF<- vector ("numeric",12) 
     for(i in 3:14) { 
     model<-summary(lm(maize[,2]~maize[,i])) 
     print (model)
     testeF[i-2] <-model$fstatistic[1]
     }
  • Incluí o efeito de dominância no modelo para ver se tem desvios de dominância:
testeF<- vector ("numeric",12) #criar um vetor numérico para 12 marcadores
     for(j in 3:14) { 
     model1<-summary(lm(maize[,2] ~ as.factor(maize[,j])))   
     print (model1)
     testeF[j-2] <-model1$fstatistic[1]
     }
  • Os valores do teste F para todos os marcadores pelo comando:
testeF
  • Usando os valores de recombinação gero um vetor com as distâncias cumulativas entre os marcadores
mapa<- 100*c(0.0, 0.357, 0.117, 0.11, 0.024, 0.222, 0.080, 0.134, 0.057, 0.215, 0.058, 0.086)
    cM<-cumsum(mapa) #estou multiplicando os valores de "M" (Morgan), por 100 (cM)
  • gráfico dos valores de F e as distâncias entre os marcadores
plot(cM,testeF)
